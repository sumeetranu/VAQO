How do I view the contents of the test database?

Once inside the Workspace view, if you selected to use the test database, you can view the contents of the test database by clicking on the Database button on the right side of the screen.

Can I change my imported database after I initially import it?

You cannot modify an imported database, but you can start over and enter your new SQL dump.

How do I use a test database after I imported a database?

In order to do that, you have to go back to the home page and start over. You can do so by clicking the "VAQO" button on the top left of the page.

How can I optimize a query I converted from SQL to Relational Algebra?

There isn't a function to optimize a query from the convert tab, but you can copy the query from the convert tab and paste it into the optimize tab to achieve the same results.

How can I run a query I converted from SQL to Relational Algebra?

There isn't a function to run a query from the convert tab, but you can copy the query from the convert tab and paste it into the run tab to achieve the same results.

Can I run or optimize a SQL query?

There isn't built-in functionality to run or optimize SQL queries, but the same results can be achieved by converting the SQL query to relational algebra in the convert tab, and pasting it into the desired tab (run or optimize) to get results.
